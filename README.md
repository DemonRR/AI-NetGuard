# AI-NetGuard

## 简介

基于DeepSeek等AI模型，实现智能代码审计、Webshell检测及流量分析研判，助力高效网络安全威胁识别与处置。

目前支持接入平台：Ollama、Siliconflow

部分界面布局借鉴于大佬：[**ChinaRan0**](https://github.com/ChinaRan0)

## 功能

- [x] 代码审计
- [x] Webshell检测
- [x] 流量分析
